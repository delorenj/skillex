# Project Cost Estimation

This skill performs infrastructure and development cost estimation for technical projects, enabling data-driven budgeting, build-vs-buy comparisons, and total cost of ownership (TCO) projections. Key features include resource-level infrastructure costing (compute, storage, network, licensing), role-based development effort estimates (FTEs, hours, hourly rates), multi-year TCO modeling, capex vs opex breakdowns, migration and integration cost lines, and contingency/risk buffers. It supports scenario and sensitivity analysis, side-by-side vendor comparisons, and generates exportable summaries, spreadsheets, and executive-ready breakdowns. Use cases include cloud migration planning, architecture choice evaluation, procurement/vendor selection, program budgeting, and ROI/business-case preparation. Core advantages are repeatable, transparent estimates, faster decision cycles, traceable assumptions, customizable templates, and the ability to integrate pricing feeds or organizational rate cards for accurate, auditable forecasts.
