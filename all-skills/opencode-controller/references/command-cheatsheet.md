## Core OpenCode commands

- Start OpenCode:
  `opencode`

- Verify the local OpenRouter-only model list:
  `opencode models openrouter --pure`

- Run one-shot work with an approved alias:
  `opencode run --model openrouter/fusion "<prompt>"`

- Session selection:
  `/sessions`

- Agent selection:
  `/agents`

- Model selection:
  `/models`
